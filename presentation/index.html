▶ Демо 1

Консольная утилита bin/tamagotchi

zsh — bitrix-hackathon-2026
$php bin/tamagotchi help
Тамагочи для кодового агента — консольная утилита.
Использование: tamagotchi <команда> [аргументы]
Команды:
setup установить хуки Claude Code в settings.json
list показать список питомцев
create завести питомца (--skill --name --type)
delete удалить питомца по id
hook:* обработчики хуков агента
help показать эту справку

Единая точка входа

bin/tamagotchi — argv-роутер и composition root. Отсюда доступны установка хуков, управление питомцами и внутренние обработчики, которые дёргает сам агент.

Скилл ↔ питомец

Каждый питомец привязан к одному скиллу — один скилл занимает один питомец. Новый рождается яйцом 🥚 с сытостью 50 и нулём использований.

  • Имя обязательно, скилл — тоже
  • Неизвестный скилл → мягкое предупреждение, а не отказ

Полноценный терминальный интерфейс

Запуск bin/tamagotchi-demo.php в терминале открывает меню на symfony/tui. Навигация стрелками ↑/↓, выбор — Enter, назад — Esc.

  • Мои питомцы · карточки состояния
  • Создать · мастер «скилл → имя → тип»
  • Удалить · с экраном подтверждения
non-TTY fallback: работает и через pipe

Хуки Claude Code

Вызвал скилл — в консоль агента прилетает карточка привязанного питомца. В конце сессии session-end подводит итог: кормёжка + стрик.

  • Главный инвариант: хук никогда не роняет агента
  • Любая ошибка → в лог, выход всегда exit 0
