# Презентация — Тамагочи для кодового агента

HTML-презентация проекта для хакатона Bitrix 2026. Один самодостаточный файл, без внешних
зависимостей (все стили и скрипты внутри `index.html`).

## Как открыть

Просто откройте файл в браузере:

```bash
open presentation/index.html      # macOS
```

или запустите локальный сервер из корня репозитория:

```bash
php -S localhost:8080 -t presentation
# затем откройте http://localhost:8080/
```

## Управление

| Клавиши | Действие |
|---|---|
| `→` / `Space` / `PageDown` | следующий слайд |
| `←` / `PageUp` | предыдущий слайд |
| `Home` / `End` | первый / последний слайд |
| `F` | полноэкранный режим |
| кнопка `↻ повторить` | заново проиграть анимацию на демо-слайде |

Внизу — точки-навигация по слайдам, счётчик и прогресс-бар.

## Структура

1. **Вступление** — название, суть, стек
2. **Команда** — *← сюда вписываются участники*
3. **Идея** — механика в трёх карточках
4. **CLI** — демо `bin/tamagotchi help`
5. **Создание** — демо `create` + `list`
6. **TUI-меню** — анимация главного меню
7. **Карточка** — состояние питомца
8. **Интеграция** — карточка-хук в консоли агента
9. **Механика** — настроение, стрик, эволюция
10. **Финал**

Демо-слайды воспроизводят **реальный вывод** приложения (собран с `bin/tamagotchi`
и `bin/tamagotchi-demo.php`), оформленный как «печатающийся» терминал вместо GIF.

## Как вписать команду

Откройте `index.html` и найдите блок:

```html
<!-- ▼▼▼ РЕДАКТИРУЙ ЗДЕСЬ: впиши имена и роли участников команды ▼▼▼ -->
```

Внутри — четыре карточки `.member`. В каждой поменяйте:

- `<div class="ava">🧑‍💻</div>` — эмодзи/аватар
- `<div class="name">Имя Фамилия</div>` — имя
- `<div class="role">Роль в команде</div>` — роль
- `<div class="who">…</div>` — за какой блок отвечал (A / B / C / продукт)

Карточек можно добавить или убрать — сетка подстроится автоматически.
